## 有什么新鲜事？

一些需要注意的事项:

1.  openshift-install 现在的版本是 4.1.something
2.  之前，openshift-install 会在完成时给出具体的登录说明。现在它只是给你密码。我的视频解决了这个问题。
3.  仪表运行在自动气象站上。当你完成集群时，你可以用一个命令轻松清除所有东西:`./openshift-install destroy cluster`
